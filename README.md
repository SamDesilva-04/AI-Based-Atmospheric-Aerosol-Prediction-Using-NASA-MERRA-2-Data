# AI-Based-Atmospheric-Aerosol-Prediction-Using-NASA-MERRA-2-Data
Atmospheric aerosols influence climate, air quality, and environmental monitoring. This project develops a machine learning model to analyze and predict aerosol concentrations using NASA’s MERRA-2 dataset, supporting improved climate modeling, pollution monitoring, and environmental forecasting.
